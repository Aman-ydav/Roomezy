# Face Models

Download pre-trained model weights here from:
https://github.com/vladmandic/face-api/tree/master/model

Required files:
- ssd_mobilenetv1_model-weights_manifest.json + shard files
- face_landmark_68_model-weights_manifest.json + shard files
- face_recognition_model-weights_manifest.json + shard files

These are ~7MB total and must be present before KYC submissions work.
The models directory is gitignored for weight files — only this README is committed.
